# Specification

## Summary
**Goal:** Add a new "Hybrid" category for businesses that serve food/drinks but aren't traditional restaurants, and add a new art studio business to the app.

**Planned changes:**
- Add "Hybrid" as a new cuisine type option in the backend
- Add a new art studio business entry at 2650 North Ave. unit 117 with Hybrid cuisine type, serving dirty sodas, twisted teas, and snacks
- Update frontend search filters to include "Hybrid" cuisine option
- Update filtering logic to support the new Hybrid category

**User-visible outcome:** Users can filter by the new "Hybrid" category to discover the art studio business that serves drinks and snacks, tagged as a hidden gem with art/crafts, date night, and family-friendly vibes.
